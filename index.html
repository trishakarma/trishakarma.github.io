Hi! I’m Trisha. I’m a C.P. Davis Scholar at Columbia University studying Computer Science (B.S.) and Economics (Minor). I love building. Whether it’s machine learning models, frontend, games or robots: I value the experiences I’ve gained working across domains.

My favorite project thus far has been my machine learning research to detect microplastics. From constructing a dataset and model to engineering a scalable, affordable device—my work was published in the peer-reviewed journal IJERSTE and won the CREST Gold Award. Projects

ML for Microplastic Detection

Driven by escalating water pollution in my city, I engineered a portable ML-based device for real-time microplastic detection. I collected and annotated water sample images, used data augmentation to scale the dataset to 5000+ images, and trained a Tensorflow model to 97% accuracy. My research was published in peer-reviewed journal IJERSTE & my Raspberry-Pi integrated device won the prestigious CREST Gold Award for innovation.

Autotune for Computer Vision

Inspired by producer Imogen Heap’s gesture-controlled music gloves, I built an audio processor that responds to hand movements. Using MediaPipe for computer vision and PSOLA, Librosa, and PyAudio for processing, I engineered a way to add autotune effects to input audio through simple hand gestures—no buttons needed.

Jungle Run

Jungle Run is a side-scrolling endless runner game. I built it in Godot using C# for the Godot Wild Game Jam. Players must dodge obstacles to survive. I implemented physics-based movement, parallax backgrounds, and increasing difficulty to create an engaging, fast-paced gameplay experience. I also put my UI/UX skills to the test by experimenting with game assets and pixel graphics.

Mt. Vesuvius Activity

I created this interactive dashboard using R’s Shiny package to explore seismic activity around Mt. Vesuvius. It visualizes over 12,000 events from 2011–2024, highlighting spatial and temporal patterns in magnitude and frequency. This project was part of the TidyTuesday create-your-own challenge!

UNO-Out Arcade Game

My friends and I built this Simon-says style arcade game using Arduino for Columbia’s Art of Engineering class. I programmed the game logic in C#, linking inputs to button presses and randomized LED sequences across 12 difficulty levels. I also worked on the soldering and assembly. 200+ students played our game at the Engineering Expo!

Full-Stack Stock Sentiment Analysis

I developed a full-stack web app that tracks stock portfolio performance using real-time market data and tweet-based sentiment analysis. Tweets are embedded with sentence-transformers and stored as vector embeddings in SingleStore. Built using React, Express, Jupyter Notebook, and SingleStoreDB with TypeScript, Python, SQL, and JavaScript.
